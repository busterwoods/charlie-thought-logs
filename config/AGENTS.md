# AGENTS.md

## Project Coding Parameters

Use these rules for all frontend changes in this repository.

1. Framework and assets
- Use Bootstrap 5 for layout and components.
- Use Bootstrap Icons for iconography.

2. Style guides
- Follow Google style guidance for HTML, CSS, and JavaScript.
- Keep code readable, consistent, and well formatted.
- Prefer clear naming and small, focused functions.

3. JavaScript architecture
- Do not use JavaScript modules for now.
- Use classic script loading (non-module scripts).
- Keep browser JavaScript compatible with this non-module approach.

4. Practical implementation notes
- Prefer semantic HTML structure.
- Keep CSS organized and avoid one-off overrides unless needed.
- Use unobtrusive JavaScript and avoid inline event handlers when possible.
- Do not use inline styles, inline scripts, internal <style> blocks, or internal <script> blocks for feature code.
- Reuse existing patterns already in this project before introducing new ones.

5. Default frontend stack for this repo
- HTML + CSS + JavaScript
- Bootstrap 5
- Bootstrap Icons
